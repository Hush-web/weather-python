# Weather Logger Script

A Python script that fetches live weather data from the wttr.in API and logs it to a CSV file with timestamps.

## Features
- Fetches current temperature, feels-like temperature, and weather description
- Works for any city
- Saves data to CSV with timestamps
- Error handling for API failures

## Requirements
- Python 3.x
- `requests` library
